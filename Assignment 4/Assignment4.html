Javascript Assignment 4

Create three different colored buttons (Red, Green, and Yellow) and one rotate button. On click of the rotate button, color of the buttons should rotate clockwise.

Requirements::

1] Create three buttons with red, green, yellow color respectively.

2] Write a text on button according to their color. For e.g.: If the color of button is "red", then the text should be "Red".

3] Create one button named as "Rotate".

4] On click of rotate button the color of three buttons should be changed. Like, the color of 2nd button should be changed to color of 1st button, the color of 3rd button should be changed to color of 2nd button, and the color of 1st button should be changed to color of 3rd button.

5] Make sure with each click of rotate button, the colors should get rotated clockwise.
