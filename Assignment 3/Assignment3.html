Assignment 3

Create three input box and named it as Min, Max and Mul respectively and also create one Button named it as "Validate".

Requirements:

1] Create three input boxes and named it as Min, Max and Mul respectively.

2] Create a button and named it as "Validate".

3] Following conditions on the input box should get satisfied.

1) Min < Max
2) Mul < Max
3) Mul <= Min
4) Min and Mul should be the factor of Max.
